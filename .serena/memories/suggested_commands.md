# Suggested Commands

| Command | Description |
|---------|-------------|
| `npm run build` | Full build pipeline (inject-version → minify → validate-types → type-check) |
| `npm run type-check` | TypeScript type check on declarations |
| `npm run validate-types` | Verify type declarations match cheat.js exports |
| `npm run minify` | Minify cheat.js → cheat.min.js |

## System Utils (Windows)
- `git` - standard git
- `dir` / `ls` - list files
